# screenshot

Capture desktop, app, window, region, or full-screen screenshots when a
tool-specific capture path is not a better fit.

## Skill

- [`screenshot`](SKILL.md): use bundled helpers for OS-level screenshots,
  preferring app/window captures over full-screen captures for desktop apps.

## Install

```bash
npx skills add Undertone0809/zee-agent-skills
```

Use the CLI prompt to select `screenshot`.
