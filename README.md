# Zee Skills

A personal collection of Agent Skills.

## Skill Collections

- [`flomo-skills`](flomo-skills/README.md): flomo memo search, create/edit,
  Markdown export, and analysis workflows.
- [`meta-skills`](meta-skills/README.md): skills for creating, extracting, and
  optimizing Agent Skills.

Open the folder README for the detailed skill list, usage notes, and install
instructions for each collection.

## Quick Start

```
npx skills add Undertone0809/zee-skills
```
