# Zee Skills

A personal collection of Agent Skills.

## Install

```bash
npx skills add Undertone0809/zee-agent-skills
```

The CLI can install everything or let you choose the specific skills you want.

## Skill Folders

- [`flomo-skills`](flomo-skills/README.md): flomo memo search, create/edit,
  Markdown export, and analysis workflows.
- [`meta-skills`](meta-skills/README.md): skills for creating, extracting, and
  optimizing Agent Skills.
- [`gstack-style-doc`](gstack-style-doc/README.md): generate technical docs in
  a gstack-inspired style.
- [`screenshot`](screenshot/README.md): capture desktop, app, window, region,
  or full-screen screenshots.

Open a folder README for the detailed skill list and usage notes.
